# Mini Blanket - Discord Bot

**Mini Blanket** is a lightweight Discord bot designed to help Minecraft server communities quickly share the server IP.  
It responds to the command `!ip` and automatically detects messages containing the word "ip", sending a stylish embedded message with the server address. Perfect for Discord servers that want to make joining the server easy and interactive for members.
